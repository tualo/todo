# 09 - Erweiterungen und Roadmap

## Geplante Erweiterungen

- Wiederkehrende Aufgaben.
- Erinnerungen und Benachrichtigungen.
- Teilen von Listen.
- Volltextsuche.

## Technische Erweiterungen

- Webhooks fuer Integrationen.
- Event-getriebene Verarbeitung fuer Side-Effects.
- Optionaler Export/Import.

## Entscheidungen

- Erweiterungen bauen auf stabiler Kernstruktur auf.
- Neue Features bevorzugt additiv einfuehren.
- Datenmodell-Aenderungen stets dokumentieren.
